React專案 Amazon-Clone
發表於2020-11-19|更新於2020-11-26|React
|字數總計:377|閲讀時長:1分鐘|閱讀量:

使用說明

運行前端

在根目錄 amazon-clone 下npm run start

運行後端

到 functions 目錄下firebase emulators:start --only functions

部署

前端

前端已經部署到 firebase 上了，可以點鏈接訪問 部署在firebase

如有修改只需重新構建，再次部署即可。npm run build && firebase deploy --only hosting

後端

由於 firebase 的 functions 功能需調整帳戶到付費方案，所以目前我沒有部署後端，運行時要在本地跑後端。如使用說明中說的那樣。後續可能會把後端單獨部署試試。

檔案結構

實現的功能

首頁

  • 首頁
  • 頂部輪播圖

登錄註冊

  • 登錄頁面

  • 登入登出

  • 郵箱密碼註冊

購物車

  • 購物車頁面
  • 添加商品到購物車
  • 從購物車移除商品

結算

  • 結算頁面
  • 用銀行卡支付
  • 訂單頁面

導航

  • 導航守衛，沒有登入不能通過鏈接訪問部分頁面

特效

  • 商品鼠標懸浮放大特效
  • 移除商品時的滑動過渡特效
